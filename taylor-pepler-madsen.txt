	1. What is Semantic HTML?

Code or tags that provide meaning to the website and eventual SEO.

	2. What is HTML used for?

Specifically used for formatting the wepage to display. Its used for organizing your webpage, giving it meaning as well as making it easliy editable and easy to navigate when styling and coding.

	3. What is an attribute and where do we put it?

Attributees follow a tag and give it additional meaning or specificity. E.g. href, width, height, src, etc.

	4. What is the h1 tag used for? How many times should I use it on a page?

	\The h1 tag is used to define the main heading of a webpage. It's best practice to use it only once per webpage, otherwise you can screw up your SEO from spamming h1 tags. Otherwise h2-h6 tags should be used.

	5. Name two tags that have required attributes

The <a> tag and the <img> tag are examples of tags that require attributes.

	6. What do we put in the head of our HTML document?

	Your important info, heading information such as a title or h1 tag, and specific information or introduction.

	7. What is an id?

An id is an attribute that makes a tag or element unique, specifically for css and javascript styling.

	8. What elements can I add an id to? 

You can add an id to all elements.

	9. How many times can I use the same id on a page?

An id can only be used once, it cannot be shared across multiple elements as it is specific to only one element or tag.

    10. What is a class?

A class is an attribute used to define one or multiple elements. Mainly used for pointing to style sheets and can be used in Javascript.

	11. What elements can I add a class to?

You can add a class to any element.

	12. How many times can I use the same class on a page? 

There is no limit to how many times you can use the same class.

	13. How do I get my link to open in a new tab?

Using the attribute target="_blank" after an href attribute will open your link in a new tab.

	14. What is the alt attribute used for?

If the image does not load or the user cannot see the image, the alt attribute is used as a discription of the image when hovering your cursor over the image.

	15. How do I reference an id?

You reference an id within a style sheet using the #your id name, or reference to it in javascript by using getElementById("your id name")

	16. What is the difference between a section and a div?

A div is more or less essential to dividing blocks of content, however, using section was introduced in HTML5 to represent specific blocks of content so you know where the header, footer, and parts of the body are.

	17. What is CSS used for?

CSS is used for styling webpages.

	19. What is the difference between a class and an id? - Give me an example of when I might use each one

A class allows you to style or modify multiple elements, whereas an id can only modify one specific element.

The parent child selector selects all the children within a parent and is written element > element selector. You would use this if you wanted to select multiple elements within a specific parent.

A pseudo selector isnt specifying a specific name of an element, instead it uses the state they are in. e.g. first, last, etc.

The box model is a method of spacing your content from eachother on the webpage for a more organized and clean look. The fix is to use box-sizing: border-box within your css, it automatically generates an element with 20px of padding and 5px of borders.
